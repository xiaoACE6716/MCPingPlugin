# MCPingPlugin
A Minecraft Server Ping Plugin For KookBC.

这是一个可以 PING Minecraft 服务器并返回可视化结果的 KOOK 机器人插件。

基于KookBC与JKook API

这个插件最初的版本来自于 ZX夏夜之风(SNWCreations)
